# рџ“Њ About the Project

**FoodieHub** is a full-stack Food Delivery Web Application developed using **Java, Servlets, JSP, JDBC, MySQL, HTML, CSS, and JavaScript**.

The application allows users to browse restaurants, explore menus, place food orders, manage their cart, and track previous orders. It follows a layered architecture using DAO classes for database operations and provides a responsive user interface.
